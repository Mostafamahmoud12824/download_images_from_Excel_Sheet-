# Download Images from Excel Sheet

A utility script that reads image URLs from an Excel (.xlsx) file and automatically downloads and saves the images to a local folder. This tool is designed to simplify bulk image downloading tasks and handle invalid or missing URLs gracefully.

## Features
- Read image URLs directly from Excel (.xlsx) files  
- Download images in bulk automatically  
- Skip empty, broken, or invalid URLs without stopping execution  
- Save downloaded images in an organized local directory  
- Simple and easy-to-configure workflow  

## Use Cases
- Product image downloading for e-commerce projects  
- Data migration and backup tasks  
- Automating media downloads from spreadsheet data  

## Requirements
- Node.js  
- npm
